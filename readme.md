# PDF Reader

![Demo Image](./Demo.png)

Main feature of this reader is to use read aloud feature for free.

### Some voices in chrome might not support text highlighting sync.

## How to Use?

- Open index.html
- Upload PDF
- Click on Read Aloud Button (Need Internet for good voices)
